### **IT Asset & License Management System – Backend**

**📌 Description**
Backend APIs and services for the IT Asset & License Management System. Built with Node.js, Express, and MongoDB.

---

### **🚀 Tech Stack**

✔ Node.js
✔ Express
✔ MongoDB (Mongoose)
✔ JWT Auth
✔ Multer uploads
✔ Nodemailer emails
✔ Cron jobs
✔ QR Code generation
✔ PDF report generation

---
### **📌 Features**

#### **Version 1.0**

✔ Asset CRUD
✔ Employee CRUD
✔ Assignment APIs
✔ History tracking

#### **Version 2.0**

✔ License management
✔ Email notifications
✔ Activity logs

#### **Version 3.0**

✔ JWT + RBAC auth
✔ Maintenance module
✔ Vendor management
✔ PDF & analytics
